## Creación del entorno virtual e instalación de Django

Para la creación del entorno virtual utilicé UV.  
UV es un gestor de paquetes en Python, moderno y veloz, el próximo estándar en lugar de pip.  
Otra herramienta de la cual me apoyé es Warp, una terminal moderna que integra la IA para autocompletado, y muchas otras funcionalidades.
